# Brevis Vera

A Rust workspace for C2PA (Coalition for Content Provenance and Authenticity) image processing with zkVM proofs.

## Overview

Brevis Vera provides image provenance verification using zero-knowledge proofs. It extracts C2PA manifests from images, applies image operations, and generates zkVM proofs to verify image authenticity and integrity.

## Crates

### Image Processor

Located in `image-processor/`, this workspace contains:

- **[lib](image-processor/lib/)**: Core C2PA library for extracting and manipulating C2PA manifests from JPEG and PNG images
- **[app](image-processor/app/)**: Host application that loads zkVM guest ELF and orchestrates processing
- **[prover](image-processor/prover/)**: CLI that generates zkVM proofs for image authenticity
- **[verifier](image-processor/verifier/)**: CLI that verifies proofs generated by the prover
- **[app-lib](image-processor/app-lib/)**: Shared library containing guest code for the zkVM
- **[server](image-processor/server/)**: Server component for image processing

### Supporting Crates

- **[image-editor](image-editor/)**: Host-side image manipulation library
- **[picture-reader](picture-reader/)**: Picture reading utilities

## Architecture

### Data Flow

1. **Prover (Host)**: Extracts C2PA manifest, applies image operations, computes file hash
2. **zkVM (Guest)**: Receives image data and operations, applies operations internally, computes semantic hash
3. **Verification**: Both prover and zkVM compute hashes that must match

### Supported Image Operations

| ID | Operation | Parameters |
|----|-----------|------------|
| 0 | Crop | x, y, width, height |
| 1 | Resize | width, height |
| 2 | ResizeAspect | max_width, max_height |
| 3 | Brightness | value (i32) |
| 4 | Contrast | value (i32) |
| 5 | Exposure | value (f32) |
| 6 | Gamma | value (f32) |
| 7 | Thumbnail | max_size |
| 8 | CropResize | x, y, width, height, target_width, target_height |

### Supported Image Formats

- JPEG (APP11 marker for C2PA data)
- PNG (c2pa chunks)

## Building

```bash
# Build all crates
cargo build --workspace
```

## Usage

### Prover

Generate a zkVM proof for an image:

```bash
cargo run --bin prover -- <input_image> <output_image> <operations>
```

### Verifier

Verify a proof:

```bash
cargo run --bin verifier -- <proof_file>
```
